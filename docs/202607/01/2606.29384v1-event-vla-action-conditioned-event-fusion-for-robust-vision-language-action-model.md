## 摘要
视觉-语言-动作模型已成为具身人工智能的重要范式。然而，现有的VLA模型通常假设光照良好且稳定的室内环境，而现实世界的具身操作可能涉及因光照变化导致的退化RGB观测，这对鲁棒的机器人操作提出了关键挑战。为解决这一问题，我们提出Event-VLA，一种事件增强的VLA框架，用于在不同光照条件下实现可泛化的操作。我们将退化的可见性下的VLA操作问题公式化为以RGB为中心的策略的实际鲁棒性问题，并引入事件流作为一种对光照鲁棒、对运动敏感的互补观测，以提高各可见性级别的鲁棒性。具体地，与直接将事件特征合并到全局语义标记空间的传统多模态融合不同，Event-VLA通过动作查询路由路径注入事件信息。它使用可学习的动作查询从VLA推理过程中提取与任务相关的语义，并通过门控交叉注意力有选择地聚合事件标记，构建事件感知的动作表示。这种设计保留了预训练的RGB-语言语义先验，同时有效利用事件信息进行鲁棒的动作预测。仿真和实际部署的实验表明，Event-VLA在正常光照下保持强大的操作性能，并在低光照退化及近乎黑暗的现实环境中提高了成功率。

## Abstract
Vision-Language-Action (VLA) models have become an important paradigm of embodied AI. However, existing VLA models typically assume well-lit and stable indoor settings, while real-world embodied manipulation may involve degraded RGB observations caused by illumination shifts, posing critical challenges for robust robotic manipulation. To address this gap, we propose \textbf{Event-VLA}, an event-enhanced VLA framework for generalizable manipulation across varying illumination conditions. We formulate VLA-based manipulation under degraded visibility as a practical robustness problem for RGB-centric policies, and introduce event streams as an illumination-robust, motion-sensitive complementary observation to improve robustness across visibility levels. Specifically, unlike conventional multimodal fusion that directly merges event features into the global semantic token space, Event-VLA injects event information through an action-query routing pathway. It uses learnable action queries to extract task-relevant semantics from the VLA reasoning process, and selectively aggregates event tokens via gated cross-attention to construct event-aware action representations. This design preserves the pretrained RGB-language semantic priors while effectively leveraging event information for robust action prediction. Experiments in simulation and real-world deployment show that Event-VLA maintains strong manipulation performance under normal lighting and improves success rates under low-light degradation and near-dark real-world settings.

## 2. 方法论：核心思想、关键技术细节、算法流程

### 核心思想
- 保留预训练 VLA 作为语义主干，将事件信息通过**动作查询路由**注入到动作通路中，而非直接合并到全局语义 Token 空间。这样既保持原有先验，又有效利用事件信息。

### 关键技术细节
1. **PREI（Physical Residual Event Integration）**  
   - 将异步事件流分解为三个通道的残差图：**瞬间（Instant）**、**显著（Salient）**、**持久（Persistent）**，分别捕捉近期运动、局部显著活动、持续轮廓。
   - 公式：  
     \[
     E^{\text{prei}}_t = [E^{\text{ins}}_t, E^{\text{sal}}_t, E^{\text{per}}_t] \in [0,1]^{H \times W \times 3}
     \]
   - 使用指数衰减活动图和时间窗口参数（窗口 40ms，τ_ins=3ms，τ_sal=10ms）。

2. **事件编码器与特征蒸馏**  
   - 将 PREI 图编码为 VLA 兼容的事件 Token：\( Y^e_t = [g^e_t; P^e_t] \)。
   - 使用冻结的 RGB 视觉教师（DINO）进行跨模态蒸馏，对齐事件特征与 RGB 特征空间（损失包括全局和 patch 级别的余弦相似度损失）。

3. **动作条件事件接口**  
   - 将可学习查询 Token 分为通用查询（Q_c）、动作查询（Q_a）、事件查询（Q_e），与 RGB、语言、本体、动作占位符一同输入 VLA 主干（但不包含事件 Token）。
   - 主干输出后，事件 Token 通过**门控交叉注意力**与 VLM 隐藏状态融合：
     \[
     Z^{\text{attn}}_t = \text{CrossAttn}(W_q Z^{\text{llm}}_t, W_k(Y^e_t+T_{\text{pos}}), W_v(Y^e_t+T_{\text{pos}}))
     \]
     \[
     Z^{\text{fused}}_t = \text{Norm}(Z^{\text{llm}}_t + \sigma(f_{\text{gate}}) \odot Z^{\text{attn}}_t)
     \]
   - 通过查询引导路由分离动作相关 Token 和事件相关 Token，分别送入 action head 和 event head。

4. **训练目标**  
   - 动作损失：L1 损失 \(\mathcal{L}_{\text{act}}\)。
   - 辅助未来事件预测损失：\(\mathcal{L}_{\text{evt}}\)（带内容掩码的 L1）和 \(\mathcal{L}_{\text{deriv}}\)（梯度一致性损失）。
   - 总损失：\(\mathcal{L} = \mathcal{L}_{\text{act}} + \lambda_{\text{evt}} \mathcal{L}_{\text{evt}} + \lambda_{\text{deriv}} \mathcal{L}_{\text{deriv}}\)（λ_evt=0.1，λ_deriv=0.3）。

### 算法流程（文字描述）
- 输入：语言指令、RGB 帧、本体状态、事件历史（40 ms窗口）
- 步骤1：事件流通过 PREI 转换为三通道残差图
- 步骤2：事件编码器生成事件 Token（不送入 VLM 主干）
- 步骤3：VLM 主干处理 RGB、语言、本体、动作占位符、查询 Token
- 步骤4：门控交叉注意力融合事件 Token 与主干隐藏状态
- 步骤5：查询引导路由分离动作表征和事件表征
- 步骤6：动作 head 预测未来动作块；事件 head（训练时）预测未来 PREI 残差图

## 6. 主要结论与发现
1. **鲁棒性显著提升**：在 LIBERO-Cross LL-Severe 下，Event-VLA 平均成功率可达 95.6%，而 π0 仅 61.9%，OpenVLA-OFT 仅 61.2%，MM-ACT 仅 69.6%。
2. **正常性能保持**：在原始 LIBERO 上，Event-VLA（96.5%）与 RGB 最强基线（96.3%）相当，且关闭事件通路后（96.2%）几乎无损失。
3. **事件表示和接口都关键**：PREI 优于时间表面；动作条件查询路由优于统一编码或适配器，且延迟更低（+2.157ms）。
4. **未来事件预测正则化有效**：加入事件头和梯度一致性损失可进一步提升成功率（从 94.8% 到 95.6%）。
5. **真实世界有效性**：在近暗条件下，Event-VLA（52.5%）远高于 π0（15%）和 OpenVLA-OFT（12.5%）。

## 7. 优点（亮点）
- **创新的事件-动作接口设计**：避免事件 Token 进入主干自注意力，通过门控交叉注意力和查询路由实现选择性注入，结构上分离语义与动作信息。
- **PREI 表示高效**：轻量级（仅 3 通道残差图），对事件流做了物理上合理的多尺度分解，可解释性好且计算成本低。
- **特征蒸馏策略**：通过预训练视觉教师对齐事件特征与 RGB 特征空间，使得事件 Token 易于融入 VLA 隐空间。
- **低延迟**：查询路由的额外延迟仅约 2.2 ms，适合在线操作。
- **双监督训练**：动作损失为主，辅助未来事件预测损失作为正则化，不需额外标注。
- **仿真与真实闭环验证**：在模拟器变体（LIBERO-Cross）和真实机器人上均验证了鲁棒性。

## 8. 不足与局限
- **模拟器依赖**：LIBERO-Cross 使用 RGB-to-Event 模拟器生成事件输入，可能无法反映真实事件相机的噪声、触发动态及标定误差。作者在真实实验中使用了真实 DAVIS 相机，但任务规模较小（4 个任务），模拟到真实的迁移仍有 gap。
- **实验覆盖面有限**：  
  - 真实世界仅 4 个任务，每种光照 10 次试验，统计显著性不足。  
  - 未在长时域（long-horizon）任务或场景光照剧烈变化（如突然开灯/关灯）下测试。  
  - 单一机器人平台（Franka）和特定事件相机（DAVIS346），泛化性未知。
- **硬件要求**：需要额外的事件相机及精密同步，增加了部署复杂度和成本。
- **退化类型有限**：主要针对光照下降（低亮、近暗），未测试其他退化（如运动模糊严重、遮挡、传感器故障等）。
- **遗漏极端情况**：当目标物体相对运动弱或对比度极低时（如论文提到的失败案例），事件响应可能不足，导致策略失效。
