## 摘要
世界动作模型（WAMs）通过联合建模视觉状态转换和机器人动作，为机器人操作提供了一种有前景的范式。然而，现有的WAMs受限于有限的时间上下文、粗略的场景级语言监督以及主要基于文本的条件设定，这阻碍了任务进度跟踪和细粒度语言-视频-动作的关联，同时限制了视觉上下文推理和跨实体迁移。在本文中，我们引入了WorldScape Policy 2.0，一种具有推理增强长短时记忆的可控WAM。其因果短时视觉记忆提供最近的观测作为DiT预填充，以保留局部交互动态；而其长短期事件记忆将历史VLM输出组织成全局历史、局部活跃和事件边界表示，以实现进度感知检索。检索到的历史增强了感知和自回归生成的规划令牌，为自主规划提供了隐式子目标条件；语义强制进一步将事件级指令语义转移到这一潜在规划路径中。为了建立细粒度的多模态可控性，我们构建了ManipEvent-5M，一个基于事件的具身预训练数据集，包含近500万个事件片段，带有对齐的动作轨迹、场景级任务指令、片段级子任务描述、目标图像和视频演示。这些设计为从高级指令进行自主规划和从细粒度文本、目标图像或视频上下文提示进行可控执行提供了统一接口。在模拟和真实世界平台上的实验表明，其在长时域自主规划、细粒度指令跟随和上下文适应方面具有卓越能力。

## Abstract
World Action Models (WAMs) offer a promising paradigm for robotic manipulation by jointly modeling visual state transitions and robot actions. However, existing WAMs are constrained by limited temporal context, coarse episode-level language supervision, and predominantly text-only conditioning, which hinder task-progress tracking and fine-grained language-video-action grounding while limiting visual-context reasoning and cross-embodiment transfer. In this paper, we introduce WorldScape Policy 2.0, a controllable WAM with reasoning-augmented long short-term memory. Its causal short-term visual memory supplies recent observations as DiT prefill to preserve local interaction dynamics, while its long short-term event memory organizes historical VLM outputs into global-history, local-active, and event-boundary representations for progress-aware retrieval. The retrieved history augments perception and autoregressively generated planning tokens, yielding an implicit subgoal condition for autonomous planning; semantic forcing further transfers event-level instruction semantics into this latent planning pathway. To establish fine-grained multimodal controllability, we construct ManipEvent-5M, an event-grounded embodied pretraining dataset containing nearly 5 million event segments with aligned action trajectories, episode-level task instructions, segment-level subtask captions, goal images, and video demonstrations. These designs provide a unified interface for autonomous planning from high-level instructions and controllable execution from fine-grained text, goal-image, or video-context prompts. Experiments in both simulation and real-world platforms demonstrate superior capabilities in long-horizon autonomous planning, fine-grained instruction following and in-context adaptation.

### 论文提出的方法论：核心思想、关键技术细节

1. **核心架构：推理增强的长短期记忆WAM**  
   - **因果短期视觉记忆（Short-Term Visual Memory）**：保留最近几帧（默认4个chunk）的VAE潜变量作为DiT的因果预填充，维持局部交互动态的时序连续性。
   - **长短期事件记忆（Long Short-Term Event Memory）**：在VLM分支中构建三类记忆视图：
     - 全局历史（Global-History）：对所有历史chunk的感知token进行注意力池化压缩，并与任务编码融合，形成整体进度总结。
     - 局部活跃（Local-Active）：保留最近`S_e`个已完成chunk的完整token，提供精细的近期上下文。
     - 事件边界（Event-Boundary）：根据相邻chunk语义变化的余弦距离，选择`S_b`个高变化点（并强制最小时间间隔Δ），保留关键子目标转换处的详细证据。
   - **记忆检索与门控融合**：当前推理隐变量`q_t`作为查询，通过交叉注意力检索上述记忆，然后经门控（sigmoid激活）决定每token的记忆注入强度，最终得到记忆增强的推理潜变量`\hat{q}_t`。
   - **隐式子目标规划**：在自主规划模式下，VLM根据高层级指令、当前观测和检索到的历史，自回归生成K个规划token（结合感知token形成`q_t`）。语义强制（Semantic Forcing）将对应子任务描述（由T5编码）作为训练时的语义目标，通过余弦相似度损失对齐`\hat{q}_t`与T5隐空间，从而将事件级语义迁移到潜规划路径中，推理时无需显式子任务文本。

2. **多模态提示与可控执行**  
   - 语言条件根据交互模式选择：**细粒度指令遵循**使用T5嵌入`c_t`直接注入DiT交叉注意力；**自主规划**使用记忆增强的VLM隐变量`\hat{q}_t`。
   - 视觉条件（目标图像、视频演示）作为持久视觉预填充token，始终保留在DiT因果注意力中，支持目标条件学习和上下文演示模仿。

3. **统一动作表示**：使用相对于动作chunk第一帧的Delta位姿（3D位置差 + 连续6D旋转表示 + 1D夹爪指令），对双机械臂为20维向量。每个实体配备专用动作编码器/解码器，负责与共享DiT的接口转换；流匹配直接在原始动作空间进行。

4. **ManipEvent-5M数据集构建**  
   - 聚合真实机器人数据（自采PiPER、AgiBot World等）、机器人模拟数据（RoboTwin 2.0、LIBERO）、UMI人机演示数据、人类手部第一视角视频（EgoDex），总计约512M帧、近5M事件片段。
   - 通过四阶段分层标注：两阶段标注（先利用运动与夹爪信号自动分割，再用VLM进行开放词汇重标注）、多层级帧采样、具身提示约束、多视角冲突消解，生成每个片段的高层任务描述、细粒度子任务描述、目标图像及跨实体视频演示对。

5. **三阶段训练策略**  
   - **Stage 1：事件级多模态WAM预训练**。在ManipEvent-5M上使用细粒度字幕、目标图像、视频演示作为条件，联合优化流匹配损失（动作+视频），建立细粒度语言-视频-动作对齐与短时视觉动态建模。
   - **Stage 2：记忆感知中间训练**。引入VLM推理分支与长短期事件记忆，只提供高层指令，利用语义强制损失对齐推理潜变量与任务层面语义，使模型学习自主进度感知与子目标推断。
   - **Stage 3：下游交互后训练**。根据具体任务激活对应交互模式（自主规划/细粒度指令/视觉提示），适应目标实体与动作分布。

   - 未公开训练算力细节（GPU型号、数量、时长），且模型规模大（5B+4B），复现门槛高。
   - 数据集ManipEvent-5M虽包含公开数据，但自采PiPER数据未明确是否开源。
3. **潜在偏差风险**：
   - 事件分割依赖运动与夹爪信号以及VLM自动标注，可能存在边界误差。
   - 语义强制使用固定T5目标（stop gradient），若T5 embedding质量有限可能限制对齐效果。
4. **应用限制**：
   - 长时记忆的窗口（默认8个历史chunk）可能无法应对极长任务（如数百步）。
   - 对高动态环境（如快速移动物体）的反应能力未测试。
   - 当前模型为离线滚动执行，未与实时反馈或重新规划紧密结合。
5. **结论的稳健性**：
   - 真实场景优势在少量试验下表现，需更大规模实测确认。
   - 消融实验仅在clean-only训练设置下进行，未在randomized设置下重复验证（表5、6基于clean-only训练）。
