## 摘要
动作条件世界模型是具身人工智能的关键组成部分，作为可扩展的策略评估器，减少了对昂贵真实世界交互的依赖。为了准确捕捉由多样化动作引发的动态，此类模型应满足三个关键目标——物理合理性(P)、动作一致性(A)和视觉保真度(V)，统称为PAV——同时保持对分布内(ID)专家演示和分布外(OOD)动作的鲁棒性。然而，现有方法主要依赖ID动作-视频对和像素级重构损失，并未明确优化PAV目标，且在专家数据之外的泛化能力较差。为解决此问题，我们提出PAVXploreRL，一种基于预训练潜在世界模型的强化学习框架，通过奖励驱动训练显式优化PAV目标。为提高动作泛化能力，我们的方法联合利用ID轨迹和噪声驱动的OOD动作探索，无需成对视频监督。实验表明，PAVXploreRL始终优于预训练基线，在基准测试中平均提升5.6%，并生成更高质量的PAV属性。作为策略评估器，它还能提供更可靠的性能估计，并减少如Ctrl-World等仅依赖专家数据的先前世界模型的高估偏差。代码：https://github.com/Social-AI-Studio/PAVXploreRL

## Abstract
Action-conditioned world models are a key component of embodied AI, serving as scalable policy evaluators that reduce reliance on expensive real-world rollouts. To accurately capture diverse action-induced dynamics, such models should satisfy three key objectives-Physical Plausibility (P), Action Adherence (A), and Visual Fidelity (V), collectively referred to as PAV-while remaining robust to both in-distribution (ID) expert demonstrations and out-of-distribution (OOD) actions. However, existing methods primarily rely on ID action-video pairs and pixel-level reconstruction losses, which do not explicitly optimize PAV objectives and generalize poorly beyond expert data. To address this, we propose PAVXploreRL, a reinforcement learning framework built on a pretrained latent world model that explicitly optimizes PAV objectives through reward-driven training. To improve action generalization, our method jointly leverages ID trajectories and noise-driven OOD action exploration, without paired video supervision. Experiments show that PAVXploreRL consistently outperforms pretrained baselines, achieving a 5.6% average gain across benchmarks and producing higher-quality PAV properties. As a policy evaluator, it also yields more reliable performance estimates and reduces the overestimation bias of prior expert-only world models such as Ctrl-World. - **背景**：大规模视频生成模型的发展使可控世界模型成为可能，但现有方法在训练时缺乏对 OOD 动作的视频监督，且 VLM 等评估器物理理解有限。
- **本文目标**：提出一种强化学习方法 PAVXploreRL，基于预训练潜在世界模型（VJEPA-2），通过奖励驱动训练显式优化 PAV 目标，同时利用 ID 轨迹和噪声驱动的 OOD 动作探索，无需 OOD 视频配对监督，从而提高世界模型的泛化性和评估可靠性。

## 2. 方法论：核心思想、关键技术细节
### 2.1 核心思想
- 基于预训练的潜在世界模型 VJEPA-2（具备强物理理解能力）作为奖励评估器，设计三组互补奖励分别对应 PAV 各目标。
- 采用 Diffusion-NFT 框架进行强化学习，避免似然计算和完整去噪轨迹存储，提高效率。
- 同时利用 ID 动作-视频对（有真值）和 OOD 动作（通过扰动得到，无视频配对）进行训练，通过伯努利采样切换奖励分支。

### 2.2 关键技术细节
- **世界模型架构**：基于 Wan2.2-TI2V-5B 视频扩散模型，添加动作编码（通过 AdaLN 注入时空路径）和帧打包（Framepack）历史表示。使用 MSE 重构损失进行监督微调（SFT），之后通过 RL 仅更新动作编码器和 LoRA 参数。
- **强化学习训练**：使用 Diffusion-NFT 框架。在 rollout 阶段，生成 G 组未来帧，用奖励函数评估得到标量奖励 r。保持两个模型快照（当前 πθ 和 EMA 更新 π_old），构造隐式正负去噪目标，用奖励权重化损失函数：
  \[
  L(\theta) = \mathbb{E}\left[ r \| \hat{x}^+_0 - x_0 \|^2_2 + (1-r) \| \hat{x}^-_0 - x_0 \|^2_2 \right]
  \]
- **奖励函数设计**：分为 ID 分支和 OOD 分支，通过伯努利采样（α_OOD 决定 OOD 比例）选择。最终奖励包含五个组件：
  - 像素级奖励（r_pixel）：MSE、SSIM、LPIPS 基于 GT 的对比。
  - VJEPA 编码器奖励（r_VJEPA-E）：使用 VJEPA 编码器测量预测帧与 GT 帧的潜在距离（仅 ID）。
  - VJEPA 预测器奖励（r_VJEPA-P）：使用动作条件 VJEPA 预测器评估无 GT 帧时的物理合理性，附加静态-运动正则化（对比零动作序列）。
  - 具身聚焦 VJEPA 奖励（r_emb VJEPA-E/P）：通过分割提取机器人区域，在原始帧和具身帧上分别计算，加权组合，增强动作一致性监督。
  - 视觉保真度奖励（r_VF）：无参考指标组合，包括 Laplacian 锐度、NIQE、流分布、时间梯度、帧间 SSIM 等。
- **VJEPA-2 训练**：在机器人数据上微调动作条件变体，采用两阶段训练：阶段 1 为标准一步预测；阶段 2 引入随机采样（原始/具身帧）和自回归多步预测（累积损失），提高长程一致性。
- **OOD 动作生成**：对非静态 GT 动作段施加三种扰动：时间偏移（±0.6s）、结构化幅度偏差（±0.15）、平滑时序噪声（σ=0.02），产生明显偏离 GT 的轨迹。

## 7. 优点
- **统一奖励函数显式优化 PAV 目标**：将物理、动作、视觉三种质量指标融合，覆盖 ID 和 OOD 场景。
- **无需 OOD 视频配对监督**：利用 VJEPA 的潜在预测能力，在 OOD 动作上仍可计算奖励，突破现有 RL 方法的局限。
- **模型无关且高效**：采用 Diffusion-NFT 避免似然计算，仅微调动作编码器和 LoRA，适用于各类预训练世界模型。
- **具身帧设计**：通过分割增强动作一致性监督，且不影响背景动态建模。
- **广泛的实验和消融**：在两个数据集、多种基线、策略评估任务上验证，消融充分。

## 8. 不足与局限
- **基础世界模型未整合多视图一致性**：当前只使用单视角，可能限制复杂场景下的物理合理性。
- **训练时间长**：尽管采用 Diffusion-NFT，长 rollout 和奖励计算仍导致 5 天 RL 训练，仅适合作为后训练方法。
- **OOD 动作生成方式有限**：仅基于三种经验扰动，未覆盖所有可能的 OOD 类型，可能遗漏某些高风险失败模式。
- **未在其他世界模型架构上验证**：论文明确“模型无关”但其实验仅在 Wan2.2-TI2V-5B 上实施，通用性尚需更多验证。
- **依赖外部分割和检测模块（GroundingDINO+SAM）**，可能引入额外偏差和计算开销。
- **策略评估实验仅 20 次交互×5 任务**，规模偏小，统计稳定性有待加强。
