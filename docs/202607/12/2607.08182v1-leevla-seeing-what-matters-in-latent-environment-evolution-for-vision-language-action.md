## 摘要
视觉-语言-动作（VLA）模型旨在将多模态输入映射到机器人动作。然而，现有大多数方法由于将所有视觉标记统一处理并基于人工选择因素进行推理，缺乏强调任务关键证据的机制且忽略了潜在因素，因此难以覆盖复杂的动态场景。为解决此问题，我们提出LEEVLA，一种关注潜在环境演化中重要信息的VLA架构，该架构显式引导模型关注信息丰富区域，同时保留潜在世界表征的结构化演化。为了识别显著且与指令相关的区域，我们引入了漂移引导动态优先级（DGDP），它结合了动态位置优先级（DPP）和语义漂移引导（SDG），以指导VLA智能体在训练过程中关注的位置。在此基础上，我们引入结构化特征流生成（SFFG），通过原型到外围（P2P）预测建模这些优先级特征在潜在空间中应如何演化，并采用互邻域对比（MC）损失以维持邻域间的拓扑一致性。DGDP和SFFG共同构成了任务感知的“何处-如何”训练框架。在VLA基准上的大量实验表明，LEEVLA持续优于先前方法，证实了显式任务证据引导和结构化潜在推理对于可扩展VLA均至关重要。我们的代码可在https://github.com/LyuQi127/LEEVLA获取。

## Abstract
Vision-language-action (VLA) models aim to map multimodal inputs to robot actions. However, most existing approaches struggle to cover complex dynamic scenarios due to treating all visual tokens uniformly and reasoning with human-selected factors, which lack mechanisms to emphasize task-critical evidence and ignore underlying factors. To address this issue, we propose LEEVLA, a VLA architecture for seeing what matters in Latent Environment Evolution that explicitly guides the model toward informative regions while preserving the structured evolution of latent world representations. To identify salient and instruction-relevant regions, we introduce drift-guided dynamic prioritization (DGDP), which combines dynamic position prioritization (DPP) with semantic drift guidance (SDG) to guide the VLA agent where to attend during training. On top of this, we introduce structured feature flow generation (SFFG), which models how these prioritized features should evolve in latent space via prototype-to-periphery (P2P) prediction, and a mutual-neighborhood contrastive (MC) loss to maintain topological consistency among neighborhoods. Together, DGDP and SFFG form a task-aware "where-how" training framework. Extensive experiments on VLA benchmarks show that LEEVLA consistently outperforms prior methods, confirming that explicit task-evidence guidance and structured latent reasoning are both crucial for scalable VLA. Our code is available at https://github.com/LyuQi127/LEEVLA.

- **背景**：已有方法（如 DreamVLA、CoT-VLA）引入显式中间推理，但其依赖外部条件辅助模型，增加了计算开销且探索能力不足。
- **动机**：提出在潜在特征空间中直接推理，利用通用视觉模型编码的多模态信息，同时自动发现“何处”应被关注、“如何”演化环境表示，以提升 VLA 的可扩展性和鲁棒性。

## 2. 方法论

### 核心思想
- **Latent Environment Evolution**：将动作视为环境状态在高维空间中演化的驱动，在潜在特征空间中对未来特征进行结构化预测。
- **“Where–How”训练框架**：
  - **Drift-Guided Dynamic Prioritization (DGDP)**：引导模型关注动态、指令相关的区域（where）。
  - **Structured Feature Flow Generation (SFFG)**：建模优先级特征在潜在空间中的结构化演化（how）。

### 关键技术细节

#### （1）Drift-Guided Dynamic Prioritization (DGDP)
- **Dynamic Position Prioritization (DPP)**：
  - 计算相邻时间步特征间的余弦相似度变化量 θᵢ = 1 − cos(vₜ,ᵢ , vₜ₊ₜ,ᵢ)，量化动态程度。
- **Semantic Drift Guidance (SDG)**：
  - 计算每个 patch 与指令 token 的最大内积作为指令相关得分 r。
  - 定义语义漂移 Δᵢ = clip((rₜ₊ₜ,ᵢ − rₜ,ᵢ)/τ, −δ, δ)，经归一化后得到引导因子 ωᵢ = exp(Δ̃ᵢ/τ)。
  - 最终权重 βᵢ = σ(ωᵢ·θᵢ)（σ 为 sigmoid），耦合动态与语义信息。

#### （2）Structured Feature Flow Generation (SFFG)
- **Prototype-to-Periphery (P2P) Prediction**：
  - 对未来多视图特征聚类，获得簇中心 μₗ。
  - 每个簇内按特征到中心的欧氏距离排序（从近到远：prototype → periphery）。
  - P2P 损失函数：L_P2P = 1/|C| Σₗ 1/|c̃ₗ| Σⱼ (α + βⱼ)·(1 − cos(v̂ₚ,ⱼ , vₚ,ⱼ))，其中 α 为全局调制因子（设为1），βⱼ为 DGDP 权重。
- **Mutual-neighborhood Contrastive (MC) Loss**：
  - 构建二阶互邻域正样本对：先取每个 anchor 的 Top-K 一阶邻居，再取一阶邻居的 Top-M 邻居作为二阶邻居；仅保留双向邻居（i 在 j 的邻居集中且 j 在 i 的邻居集中）作为正样本。
  - 使用 InfoNCE 损失 L_MC 拉近正样本，抑制聚类噪声导致的虚假链接。
- **总体训练目标**：L_total = λ₁L_action + λ₂L_P2P + λ₃L_MC，其中 L_action 为动作 chunk 的 L1 回归损失。

## 6. 主要结论与发现

- LEEVLA 在两个规模（mini 和 large）上均达到 SOTA 性能：
  - LIBERO：LEEVLA-large 平均 98.2%，LEEVLA-mini 97.5%，均优于 π0.5 (97.5%) 和 OpenVLA-OFT (97.1%)。
  - CALVIN：LEEVLA-large 平均任务长度 4.34，优于 π0.5 (4.02) 和 OpenVLA-OFT (4.10)。
  - 真实世界：LEEVLA-large 平均成功率 78.5%，远高于 OpenVLA 的 40%。
- 显式任务证据引导（DGDP）和结构化潜在推理（SFFG）对可扩展 VLA 至关重要。
- 潜在空间推理比人工选择条件更灵活、更鲁棒，且训练时额外开销不影响推理效率。

## 7. 优点

- **创新性**：首次将“where–how”框架引入 VLA，DGDP 自动识别动态、指令相关区域，SFFG 通过原型到外围预测保持空间语义连续性。
- **训练-推理解耦**：DGDP 和 SFFG 仅在训练时使用，推理时无任何额外计算或内存负担，可轻松集成到现有 VLA 架构。
- **多尺度验证**：在 0.5B 和 7B 两种规模上均取得 SOTA，方法具有通用性。
- **消融全面**：逐步消融明确了每个模块的贡献，并分析了特征重排序、全局因子等设计选择。
- **真实世界验证**：在实际机器人平台上展示了有效性，包括未在训练中见过的场景。

## 8. 不足与局限

- **实验覆盖面**：
  - 真实世界任务仅 3 个，且仅使用单一种类机械臂（UR5），缺乏跨具身、跨环境的泛化验证。
  - 未在更多样化的数据集（如 BridgeData、RoboNet）上评估，可能限制结论通用性。
- **潜在偏差风险**：
  - DGDP 权重依赖于预训练视觉特征（DINOv2、SigLIP），这些特征本身可能存在针对某些物体的偏向。
  - MC 损失的互邻域选取可能对聚类质量和阈值（K、M）敏感，论文未充分讨论参数影响。
- **计算开销**：
  - 训练时 MC 损失延迟高达 1200 ms（表 4），可能导致训练时间显著增加（论文未报告总训练时间）。
  - 需要额外 GPU 显存进行特征排序和邻域搜索，对资源有限的研究者不友好。
- **可解释性**：潜在空间推理虽避免了人工条件，但可解释性低于显式视觉推理（如生成子目标图像）。
- **方法假设**：假设任务相关信息可以通过动态性和指令相关性捕捉，但某些复杂任务可能需要长程时间依赖，单一帧的静态对比可能不足。
