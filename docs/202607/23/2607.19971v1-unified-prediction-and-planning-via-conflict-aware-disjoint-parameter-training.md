## 摘要
周围智能体的准确运动预测与安全运动规划是社会机器人在拥挤环境中导航的两个紧密耦合的关键任务。在资源受限的边缘设备上部署这些系统需要能够同时执行这两个任务的紧凑统一模型。然而，在这些紧凑共享编码器中，近期统一模型常常忽略了预测邻居行为与以自我为中心的安全规划这两个不同目标所带来的严重表征冲突。为解决此问题，我们首先识别出技能冲突——一种参数重叠分配导致不同任务竞争相同权重的现象，阻碍模型完全专注于单个技能。为此，我们提出了一种基于模型合并的新框架——分离参数训练（DPT）。DPT通过分布式参数学习缓解技能冲突导致的性能下降，在合并前分离每个任务的关键参数区域，同时保留其核心能力。此外，我们观察到稀疏合并（即选择性整合每个任务中最具影响力的参数，而非合并所有任务特定参数）通过防止相邻特征间的干扰并集中表征能力，实现了最优性能。DPT可与多种合并方法并行应用。在标准人群导航基准（JRDB和JTA）上的评估表明，我们的框架展现出优越性能，验证了其在安全、资源高效的机器人导航中的通用性和有效性。

## Abstract
Accurate motion prediction of surrounding agents and safe motion planning are two closely coupled key tasks for social robot navigation in crowded environments. Deploying these systems on resource-constrained edge devices necessitates compact, unified models that can perform both tasks simultaneously. However, within these compact shared encoders, recent unified models often overlook severe representational conflicts that arise from the distinct objectives of predicting neighbor behaviors versus ego-centric safety planning. To address this issue, we first identify the Skill Conflict$\unicode{x2014}$a phenomenon where overlapping parameter assignments cause distinct tasks to compete for the same weights, preventing the model from fully specializing in individual skills. To resolve this, we propose a novel model-merging-based framework, Disjoint Parameter Training (DPT). DPT mitigates performance degradation caused by Skill Conflict through distributed parameter learning, which separates the key parameter regions of each task while preserving their core capabilities prior to merging. In addition, we observe that sparse merging, which selectively integrates only the most influential parameters for each task rather than combining all task-specific parameters, yields optimal performance by preventing interference among adjacent features and concentrating representational capacity. DPT can be applied in parallel with a variety of merging methods. Evaluated on standard crowd navigation benchmarks (JRDB and JTA), our framework demonstrates superior performance, validating its versatility and effectiveness for safe, resource-efficient robot navigation.

#### 1. 核心问题与整体含义（研究动机和背景）
- **问题背景**：社会机器人在拥挤环境中的导航需要同时进行**运动预测**（预测周围智能体的未来轨迹）和**运动规划**（生成自身安全轨迹）。传统顺序式流水线存在延迟高、无法联合推理的缺点。近期研究倾向于使用统一模型（一个共享编码器+两个任务头）同时处理两个任务，以满足边缘设备（如机器人）的资源受限要求。
- **核心问题**：在紧凑的统一模型中，预测任务和规划任务的目标本质上存在冲突——预测需要捕捉周围所有智能体的多样行为，而规划需要关注以自我为中心的安全和可行性。两者在共享编码器中竞争相同的参数，导致“**技能冲突（Skill Conflict）**”——一种参数重叠引发任务相互干扰的现象，严重降低两个任务的性能。
- **研究意义**：解决技能冲突是实现紧凑、高效、安全的统一预测-规划模型的关键，尤其适用于资源受限的自主导航系统。

#### 2. 方法论：核心思想、关键技术细节
- **核心思想**：采用**模型合并（Model Merging）** 范式替代传统联合训练，先为每个任务独立微调出“材料模型”，再合并为一个统一模型。为了避免微调过程中参数密集重叠，提出**分离参数训练（Disjoint Parameter Training, DPT）** 和**稀疏合并（Sparse Merging）**。

- **关键技术细节**：
  1. **技能冲突分析**：通过任务向量（Task Vector）的激活掩码观察，当两个任务各自利用较多参数（比如前50%）时，重叠比例高达39.25%（Fig. 3），导致联合性能上限下降。
  2. **DPT算法**（Algorithm 1）：
     - 从预训练模型 `Θ₀` 出发，初始化两个任务各自的二进制掩码 `M_plan = 0, M_pred = 0`。
     - **第一阶段（前E/2个epoch）**：交替处理两个任务。在每个任务步骤，计算当前损失和梯度，从未被任何任务占用的参数中选出**梯度绝对值最大的 top-K 个参数**，将其永久分配给当前任务（掩码置1），并仅更新这些参数。掩码逐步增长。
     - **第二阶段（后E/2个epoch）**：冻结掩码，仅更新已分配的参数，使每个任务的核心技能在固定区域巩固。
     - 结果：两个任务的材料模型具有**互不相交**的激活参数区域。
  3. **稀疏合并**：即使参数区域不重叠，相邻坐标或层之间的特征仍可能相互干扰。因此，合并时仅每个任务向量的**前K%最大幅度坐标**（实验中 K=1%~2%最佳），定义合并模型为：  
     `Θ_merged = Θ₀ + M_pred^(K%) ⊙ τ_pred + M_plan^(K%) ⊙ τ_plan`  
     其中τ是任务向量（微调后参数减预训练参数）。

- **可选增强**：在合并后，还可以在未激活的参数区域进行联合推理微调（Joint Reasoning），进一步利用剩余容量促进跨任务协同。

#### 3. 实验设计
- **数据集与场景**：

  - **JTA**（大规模合成数据集）：含丰富标注（2D/3D关键点、边界框等），同样使用9帧输入预测12帧。

#### 6. 主要结论与发现
- **技能冲突是紧凑统一模型中真实且显著的性能瓶颈**：当两个任务利用超过一定比例的参数时，重叠区域导致联合性能天花板降低。
- **DPT有效缓解技能冲突**：通过分离微调的参数空间，DPT使得各任务核心技能紧凑地存储在不相交区域，合并后性能大幅提升（例如规划ADE从0.9387降至0.4044，碰撞率显著降低）。
- **稀疏合并进一步提高性能**：仅使用 top-1%~2% 的任务向量，比全参数合并或中度稀疏合并效果更好，说明相邻特征干扰依然存在。
- **DPT可泛化到多种合并方法**：与 Task Arithmetic、TIES-Merging、T-Switch 结合均带来提升。
- **联合推理阶段可互补**：在分离参数外剩余区域进行联合微调，可进一步利用协同作用。
- **在 JRDB 和 JTA 上均取得最优性能**，且推理延迟仅10.4ms，适合边缘部署。

#### 7. 优点
1. **问题识别精准**：首次系统性地在紧凑统一模型中识别并量化“技能冲突”现象，为后续研究提供基础。
2. **方法创新且简洁**：DPT 通过交替选择 top-K 梯度参数实现任务参数空间天然分离，不引入复杂结构，对原骨干改动小。
3. **通用性强**：DPT 可作为独立模块与几乎所有模型合并方法组合，并在两个领域的任务（行人导航、自主驾驶）中验证有效。
4. **性能提升显著**：相比现有基线（包括统一模型 DIPP/DTPP、合并方法 Task Arithmetic 等），在规划和预测各指标上均取得大幅领先（例如规划ADE从~0.80降至~0.40，碰撞率从~1.5%降至~0.9%）。
5. **推理高效**：合并后模型大小6.44M参数，1.65 GFLOPs，延迟10.4ms，适合边缘设备。

#### 8. 不足与局限
1. **训练成本增加**：DPT 需要额外步骤（交替分配和冻结阶段），GPU-hours约为普通联合训练的2.6倍（4.10 vs 1.58），但作者指出推理成本无影响。
2. **最佳超参数依赖任务平衡**：DPT分配比例需要根据任务优先级手动选择（如规划优先时偏向规划更多参数），缺乏自适应机制。
3. **稀疏合并的K值敏感**：K=1%~2%最佳，但K稍大（如5%）性能开始下降，可能需要在不同数据集上微调。
4. **仅测试了特定架构**：主实验以 Social-Transmotion + 规划解码器为骨干；虽在 DTPP 和 HiP-AD 上额外验证，但未测试更多异构架构（如纯Transformer、扩散模型等）。
5. **理论分析有限**：技能冲突的定义基于参数重叠的观测，未提供严格的数学论证其必然性与下界。实验上通过控制重叠验证因果关系，但仍属经验性。
