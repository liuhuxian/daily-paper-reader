## 摘要
DreamerV3系列的基于模型的强化学习智能体在任务序列上训练时会灾难性地遗忘，即使无界回放缓冲区保存了所有早期经验。我们提出了持续强化学习文献中假设已得到答案但从未测量过的问题：哪个组件遗忘？在不清除回放的情况下，预注册的组件级探针（全程n=3种子）显示，世界模型基本上保留了旧任务的所有可测量内容——奖励区分（保留率~1.0）、价值估计和终止结构——而演员的行为则崩溃。此设置下的遗忘是通道问题，而非记忆问题。我们通过干预证明了这一点：冻结世界模型并使用相同的想象轨迹，想象中的强化学习无法恢复丢失的技能（0/3种子），而在世界模型自己的分级梦境上进行监督自我模仿，在零环境交互的情况下以3/3种子恢复了技能。在训练期间交错进行，这种分级梦境排练产生了一种无需任务标签、参数恒定的持续学习器：3/3的四任务链保留（普通回放通过率为0/3），3/3的八任务链保留，并且与匹配的真实情节克隆相比一致增益（配对差异+0.13，自助法95%置信区间[0.07, 0.24]，完全种子分离）。梦境分级步骤是关键：我们描述了两种评分失败模式，提供了一个离线选择度量，在它们污染结果之前捕获了它们，并给出了一个实现在先的分级规则来关闭它们。所有实验均采用承诺协议进行预注册；每个被反驳的假设都被报告。

## Abstract
Model-based reinforcement-learning agents of the DreamerV3 family forget catastrophically when trained on task sequences, even when an unbounded replay buffer preserves every earlier experience. We ask a question the continual-RL literature has assumed an answer to but never measured: which component forgets? Under never-clear replay, pre-registered component-level probes (n=3 seeds throughout) show that the world model retains essentially everything measurable about old tasks -- reward discrimination (retention ratio ~1.0), value estimates, and termination structure -- while the actor's behavior collapses. Forgetting in this regime is a channel problem, not a memory problem. We demonstrate this by intervention: with the world model frozen and identical imagined rollouts, reinforcement learning in imagination fails to recover a lost skill (0/3 seeds), while supervised self-imitation on the world model's own graded dreams recovers it on 3/3 seeds with zero environment interaction. Interleaved during training, this graded dream rehearsal yields a task-label-free, parameter-constant continual learner: 3/3 four-task chains retained where plain replay passes 0/3, 3/3 eight-task chains, and consistent gains over matched real-episode cloning (paired difference +0.13, bootstrap 95% CI [0.07, 0.24], complete seed separation). The dream-grading step is load-bearing: we characterize two scoring failure modes, provide an offline selection gauge that caught both before they contaminated results, and give a realized-first grading rule that closes them. All experiments were pre-registered with committed protocols; every refuted hypothesis is reported.

## 1. 核心问题与整体含义（研究动机与背景）

- **问题**：DreamerV3 系列基于模型的强化学习（MBRL）智能体在连续任务学习过程中会发生灾难性遗忘，即使使用了**无界回放缓冲区**（保留所有早期经验）也无法避免。现有持续 MBRL 方法（如 Continual-Dreamer、WMAR、ARROW）隐含假设：保护世界模型就能连带保存策略性能，但该假设从未被组件级验证。
- **核心发现**：作者首次通过组件级探针定位遗忘根源——**世界模型保留几乎所有旧任务知识（奖励、价值、终止结构），而演员（actor）的行为策略崩溃**。遗忘因此被重新定义为**通道问题**（policy-gradient channel 无法将完整信号转化为保留的行为）而非记忆问题。
- **整体含义**：持续 MBRL 的现有保护范式应转向直接修复演员的学习通道，而非过度保护世界模型。

## 2. 方法论：核心思想、关键技术细节

### 核心思想
- **不依赖参数隔离**：单演员、单世界模型，无需任务标签、无需增加参数。
- **利用世界模型作为生成器**：从先前任务的缓冲状态出发，用当前（采样）演员在冻结或实时世界模型中想象轨迹，然后通过**监督自模仿**（行为克隆）训练演员，而非策略梯度。
- **分级（grading）是关键**：对想象轨迹进行评分，选择高质量轨迹进行克隆，并设计“实现在先”规则避免评分偏差。

### 关键技术细节
1. **梦境排练（Dream Rehearsal）算法（Algorithm 1）**：
   - 每个任务阶段结束后，保留其缓冲经验作为“排练起始状态”。
   - 在新任务训练中，每 2000 个环境步骤后，对每个先前任务执行 50 次梦境自模仿更新：
     - 从该任务的起始状态采样，用当前演员在想象中滚动 15 步（horizon H=15）。
     - 用实时世界模型（奖励头、终止头、价值头）对每条想象轨迹评分。
     - 行为克隆得分最高的 25% 轨迹（top-q=0.25）。
   - 世界模型和评论家（critic）正常训练，不受干扰。
   - 计算开销：任务数线性增加（4 任务时约 +15%）。

2. **实现在先评分规则（Realized-first grading, Eq.1）**：
   ```markdown
   让 c_t 为终止头概率，p_t = ∏_{k<t} c_k 为轨迹存活概率，r_t 为预测奖励，V 为评论家，γ=折扣。
   realized = Σ_{t=0}^{H-1} γ^t p_t r_t
   score = 10·1[realized > 0.3] + realized + γ^H p_H V(s_H)
   ```
   - **关键改进**：首先判断想象中是否实际获得奖励（realized > 0.3），超过所有仅依赖价值承诺的轨迹；其次用存活概率 p_t 加权，避免后终端污染（post-terminal contamination：梦境超出终止点后 bootstrap 噪声）；最后内部排序基于真实奖励 + 终止敏感的引导值。
   - **离线选择度量**：作者附带一个直接测量评分器质量的指标（AUC、top-quartile purity），在实验前即捕获了两种失败模式（后终端污染和承诺高估）。

3. **修复通道问题的证据**（第5节隔离实验）：
   - 冻结整个训练后的世界模型，仅从该模型想象数据，重训练漂流演员。
   - **RL-in-imagination**（标准 DreamerV3 actor-critic 更新）：0/3 种子恢复。
   - **梦境自模仿**（监督克隆 top-25% 分级想象轨迹）：3/3 种子恢复，零环境交互。

## 3. 实验设计

### 环境与任务
- **MiniGrid**（64×64 RGB 部分观测），使用 DreamerV3 参考实现（17M 参数世界模型，1.8M 参数演员）。
- **核心任务链（4任务）**：DoorKey-5x5 → SimpleCrossingS9N1 → LavaGapS5 → MultiRoom-N2-S4。包含不同 episode 长度（~10 到 ~80 步）以及致死性任务。
- **扩展链（8任务）**：附加 LavaCrossingS9N1、DistShift2、DoorKey-6x6、Unlock。
- **无界回放缓冲区**：所有经验累积保存（Continual-Dreamer 基线方案）。
- **评估**：每 2000 环境步骤一次，每个已学任务运行 10 个真实 episode。通过标准：所有任务最后 3 轮均值 ≥0.6 回报。
- **phase 推进条件**：所有已学任务连续 3 轮评估 ≥0.6，或达到 150k 步上限。

### 对比方法
1. **普通无界回放（Plain replay）**：等同于 Continual-Dreamer 基线。
2. **冻结演员+路由器（Frozen heads + router）**：存储每个任务训练好的演员快照 + 最近中心分类器（任务不可知评估），作为隔离参考上限。
3. **真实 episode 克隆（Real-episode cloning）**：从缓冲区直接克隆高回报 episode（return>0.05）的 action，与梦境排练等更新次数。
4. **RL-in-imagination vs. 梦境自模仿**（第5节隔离比较）。
5. **消融评分规则**：两种失败模式（naive scorer、survival-weighting）对照实现在先规则。

### 指标
- **最终保留（Final retention）**：每个任务最后 3 轮评估均值。
- **奖励区分度（Reward discrimination）**：世界模型奖励头在成功/零奖励步上的预测差异。
- **AUC / top-quartile purity**：用于离线验证评分器质量。

### 种子
- 所有实验 n=3（部分实验额外增加），报告每个种子值及分散度（不单独依赖 pass 计数）。

## 6. 主要结论与发现

1. **定位遗忘根源**：在无界回放的 MBRL 中，世界模型保留旧任务知识（奖励区分保留率 ~1.0，价值高，终止准确），而演员行为崩溃。**遗忘是通道问题，而非记忆问题**。
2. **策略梯度通道失败**：相同想象数据下，RL-in-imagination 无法恢复技能（0/3），但监督自模仿恢复（3/3），证实 channel 本身不稳定。
3. **分级梦境排练有效**：4 任务链 3/3 种子全部保留（普通回放 0/3），8 任务链 3/3 全部保留。
4. **优于真实 episode 克隆**：梦境排练在困难任务保留上显著优于过滤克隆（配对差 +0.13，95%CI [0.07,0.24]，完全种子分离）。
5. **评分规则是关键**：实现在先规则（Eq.1）关闭了后终端污染和承诺高估两种失败模式，使选择质量 AUC=1.0。
6. **缩放性**：8 任务无额外参数、无任务标签，计算成本线性增长（因早期任务加速获得补偿），通过后未见稀释现象。

## 7. 优点

- **现象新颖且清晰**：首次组件级测量持续学习中的遗忘来源，推翻保护世界模型的隐含假设。

## 8. 不足与局限

- **领域局限**：仅在 MiniGrid 网格世界（64×64 RGB）验证，world model 仅 17M 参数。作者声明显式承认：“whether the pattern persists in larger domains is an open empirical question”。
- **任务顺序固定**：每链长度下只有一种顺序，未测试顺序敏感性。
- **种子数量少**：n=3，虽然报告了种子分离，但统计稳健性有限（尤其对于随机性大任务）。
- **计算成本线性增长**：在更多任务时可能达到计算瓶颈（当前 8 任务无问题，但未测更大）。
- **依赖保留原生经验**：排练起始状态来自缓冲区，未测试生成式起点的替换（未来工作）。
- **无法处理探索限制**：对于无法在贪婪探索下启动的任务（长期稀疏任务），未提供解决方案（列为 acquisition frontier）。
- **评分规则负载关键**：若评分器再出现未预见的失败模式，方法可能失效（作者通过离线度量缓解）。
- **对比方法覆盖面**：只对比了真实 episode 克隆，未对比近期持续 MBRL 方法（如 WMAR、ARROW）在演员训练上的完整扩展（但作者指它们与自身方法正交）。
