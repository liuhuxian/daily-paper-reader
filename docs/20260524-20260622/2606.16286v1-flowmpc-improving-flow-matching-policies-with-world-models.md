## 摘要
流匹配（Flow Matching, FM）是多模态动作空间中行为克隆的一种强大方法 [Jiang et al., 2025]，但由于其训练目标并非直接最大化期望回报，因此在测试时 FM 策略的表现仍有改进空间。本文研究学习到的世界模型是否能够通过使策略提出的候选动作序列进行模型预测路径积分（Model Predictive Path Integral, MPPI）规划来改进 FM 策略。基于 TD-MPC2 [Hansen et al., 2024]，我引入了 FlowMPC，这是一个将模仿学习的 FM 策略与学习到的世界模型相结合的框架，用于 ManiSkill 操作任务 [Tao et al., 2025] 中的测试时规划。在 PickCube 和 PickSingleYCB 任务中，添加世界模型比单独使用 FM 策略提升了性能，尤其在回合末成功率方面提升显著。这些结果表明，基于世界模型的规划可以有效补充基于流的模仿策略，而无需修改 FM 训练目标。

## Abstract
Flow Matching (FM) is a powerful approach for behavior cloning in multimodal action spaces [Jiang et al., 2025], but because it is not trained to directly maximize expected return, there is still room to improve how FM policies act at test time. This work investigates whether a learned world model can improve FM policies by enabling Model Predictive Path Integral (MPPI) planning over candidate action sequences proposed by the policy. Building on TD-MPC2 [Hansen et al., 2024], I introduce FlowMPC, a framework that combines an imitation-learned FM policy with a learned world model for test-time planning in ManiSkill manipulation tasks [Tao et al., 2025]. Across PickCube and PickSingleYCB, adding the world model improved performance over the FM policy alone, with especially clear gains in end-of-episode success. These results suggest that world-model-based planning can effectively complement flow-based imitation policies without modifying the FM training objective.

## 2. 论文提出的方法论

### 核心思想
- 使用 FM 策略生成多模态候选动作序列，再利用 TD-MPC2 风格的世界模型进行模型预测路径积分（MPPI）规划，从候选轨迹中选出预期回报最高的动作并执行。

### 关键技术细节
- **训练流程分五阶段**：
  1. 训练 SAC 专家收集演示数据；
  2. 收集完整轨迹（RGB、状态、动作、奖励）；
  3. 训练 FM 策略（条件 1D 时序 U-Net，输出动作序列）；
  4. 使用 FM 视觉编码器预计算紧凑视觉潜变量 \( z_{\text{viz},t} \)，与 proprioceptive state 拼接得到世界模型输入 \( x_t = [s_t, z_{\text{viz},t}] \)；
  5. 在线训练世界模型，replay buffer 预加载专家数据。
- **世界模型**：基于 TD-MPC2，包含动力学模型、奖励模型、值函数（Q 函数）和行为克隆预测器 \( \pi_{\text{BC}} \)。
- **关键修改**：
  - 用 FM 策略替换 TD-MPC2 原始的 actor；
  - 添加行为克隆头 \( \pi_{\text{BC}} \) 用于终端值自举；

- **训练损失**（以状态空间版本为例）：
  - 动力学损失：\( \mathcal{L}_{\text{dyn}} = \|\hat{x}_{t+1} - x_{t+1}\|_2^2 \)
  - 奖励损失：\( \mathcal{L}_{\text{rew}} = \text{BCEWithLogits}(\hat{r}_t, r_t) \)
  - Q 值损失：\( \mathcal{L}_Q = \sum_{i=1}^{N_Q} \|Q_i(x_t, a_t) - y_t\|_2^2 \)，其中 \( y_t = r_t + \gamma \hat{Q}_{\text{targ}}(x_{t+1}, \pi_{\text{BC}}(x_{t+1})) \)

## 6. 论文的主要结论与发现

- **主要结论**：将学习到的世界模型与 MPPI 规划结合到 FM 策略的测试时推理中，可以显著提升回合末成功率：PickCube 上从 93.14% 提高到 97.44%，PickSingleYCB 上从 56.81% 提高到 66.41%。
- **发现**：
  - 提升主要体现在“保持成功状态到回合末”而非“短暂达到成功”（end success 增益大于 anytime success）。
