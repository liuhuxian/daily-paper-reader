## 摘要
世界模型正从被动的视觉生成器转变为物理人工智能的基础性、可操作基础设施：它们必须原生地从异构经验中获取世界知识，在长时域上维持持久状态，并在实际部署约束下高效执行。我们提出了Kairos，一个围绕这些需求设计的原生世界模型栈。（1）Kairos通过首创由跨具身数据课程指导的原生预训练范式来学习世界，该课程将开放世界视频、人类行为数据和机器人交互组织成渐进式发展路径。（2）Kairos通过配备混合线性时间注意力的原生统一架构实现统一的世界理解、生成与预测，其中滑动窗口注意力捕捉局部动态，扩张滑动窗口捕捉中程依赖，门控线性注意力维持持久全局记忆。我们建立了形式化理论界限，证明这种时间因子分解严格限制了误差累积，从数学上保证了跨扩展时域的状态传播。（3）Kairos通过引入部署感知的系统协同设计来运行世界，支持在服务器和消费级硬件上为真实观察-动作-反馈循环生成低延迟推出。在具身世界模型、长时域和动作策略基准上的实验表明，Kairos在取得顶级性能的同时，提供了强大的效率与能力权衡。这些结果共同将Kairos定位为未来自我进化物理智能的凝聚性操作基础。

## Abstract
World models are transitioning from passive visual generators to foundational, operational infrastructure for Physical AI: they must natively acquire world knowledge from heterogeneous experience, maintain persistent states over long horizons, and execute efficiently within real deployment constraints. We introduce Kairos, a native world model stack designed around these requirements. (1) Kairos learns the world by pioneering a Native Pre-training Paradigm governed by a Cross-Embodiment Data Curriculum, which organizes open-world videos, human behavioral data, and robot interactions into a progressive developmental pathway. (2) Kairos maintains the world by unified world understanding, generation, and prediction within a Native Unified Architecture equipped with Hybrid Linear Temporal Attention, where sliding-window attention captures local dynamics, dilated sliding windows capture mid-range dependencies, and gated linear attention maintains persistent global memory. We establish formal theoretical bounds demonstrating that this temporal factorization strictly limits error accumulation, mathematically guaranteeing state propagation across extended horizons. (3) Kairos runs the world by incorporating a Deployment-Aware System Co-Design to support low-latency rollout generation on server and consumer-grade hardware for real-world observation-action-feedback loops. Experiments on embodied world-model, long-horizon, and action-policy benchmarks show that Kairos achieves top level performance while offering a strong efficiency-capability trade-off. Together, these results position Kairos as a cohesive operational foundation for future self-evolving physical intelligence.

## 2. 方法论：核心思想、关键技术细节

Kairos 围绕三个支柱构建：

### 2.1 原生预训练范式与跨具身数据课程（CEDC）

- **核心思想**：拒绝后训练微调，而是将物理和具身知识从零开始原生注入。
- **三阶段预训练**：
    - **阶段I：物理预训练**。使用开放世界视频学习物理规律（重力、碰撞、物体永久性）。采用渐进式训练：图像预训练（256p）→ 图像-视频混合预训练（256p→480p→720p）→ 持续预训练（最长241帧）。引入形状感知时间步重排（Shape-aware timestep shift）。
    - **阶段II：人类中心具身预训练**。使用人类演示和部分机器人数据，学习行为语义和任务结构化意图。训练目标仅优化 VideoDiT，逐步过渡到指令式标题。
    - **阶段III：联合世界-动作训练**。同时优化 VideoDiT 和 ActionDiT（约1/5参数），采用流匹配损失 L = L_video + λ L_action。动作预测分支可独立于视频生成进行高效推理。
- **训练目标**：基于流匹配（Flow Matching），条件速度场回归。

### 2.2 原生统一架构：理解-生成-预测 + 混合线性时间注意力

- **整体架构**：Mixture-of-Transformers（MoT），三个模块：
    - **World Understanding**：VLM（Qwen系列）提取语义表示。
    - **World Generation**：基于 Diffusion Transformer（DiT）进行条件视频生成。采用线性复杂度混合注意力。
    - **World Prediction**：联合建模环境动态和机器人动作。统一注意力掩码：历史视频 token 互关注，未来视频和动作 token 关注所有历史，未来动作 token 采用全注意，未来视频 token 采用稀疏时空注意。
- **混合线性时间注意力（Linear DiT）**：
    - **滑动窗口注意（SWA）**：局部动态，窗口大小 = L × window_size。
    - **扩张滑动窗口注意（DSWA）**：中程依赖，引入扩张因子。
    - **门控线性注意（GLA）**：基于GatedDeltaNet，采用Delta更新规则和门控机制，作为持久因果记忆，线性复杂度。
- **理论证明**：
    - **必要性（Theorem 1）**：如果最优全历史预测器不可从近期窗口恢复，则纯局部注意必然产生严格正的非可减多余风险。
    - **充分性（Theorem 2）**：若Bayes预测器可分解为共享状态、短程、中程和全局记忆分支，且全局记忆采用收缩门控更新（ρ<1），则混合多尺度记忆可近似充分，多余风险受限于局部近似误差和全局记忆扰动。

### 2.3 部署感知系统协同设计

- **时间步蒸馏**：将480P教师模型蒸馏为4步学生生成器，采用分布匹配蒸馏（DMD）+ 一致性蒸馏（CM）混合目标。
- **硬件感知推理优化**：
    - **云服务**：混合并行（序列并行+张量并行），滑动窗口采用Ulysses并行，Cross Attention采用序列并行，GLA采用修改张量并行。使用TeaCache、Torch.compile、内核融合。
    - **消费级设备**：FP8低精度计算（Q/K用INT8/INT4，P·V用FP8），SmoothQ，逐线程/逐warp量化。权重量化Int4文本编码器。Tile-based Gated DeltaNet流式访问。
- **效率对比**：Kairos-4B在720P 5秒生成中仅需43秒（单GPU A800），9秒（4GPU），远优于Cosmos-Predict2.5-14B（2526/687秒）、Wan2.2-5B（201/85秒）。推理时间随视频长度线性增长。
