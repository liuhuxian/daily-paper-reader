## 摘要
在现实世界中进行稳健的机器人操作不仅需要理解当前观测，还需要记忆和动力学建模。世界动作模型（WAM）通过联合建模视觉预测和基于当前及历史观测的动作条件具备了这些能力，使其成为机器人操作的一种有前景的范式。然而，现有的WAM面临一个基本权衡：高效推理的方法通常仅以最近观测的有界窗口为条件，因此在非马尔可夫环境中表现不佳；而保留长历史的方法则随着序列长度增加，时间和空间成本显著增长。为解决这一挑战，我们提出了MemoryWAM，一种具有高效持久记忆的世界动作模型。MemoryWAM采用混合记忆设计，结合了最近的帧、事件边界锚定帧以及总结长程历史的紧凑要点标记。定制的注意力机制能够检索详细的短期上下文和压缩的长期上下文，支持依赖记忆的决策，同时减少推理延迟和GPU内存使用。在模拟和现实世界中的长时域、依赖记忆的操作任务中，MemoryWAM优于强大的视觉-语言-动作（VLA）和WAM基线，同时保持了良好的计算效率。

## Abstract
Robust robotic manipulation in the real world requires not only an understanding of the current observation, but also memory and dynamics modeling. World action models (WAMs) possess these capabilities by jointly modeling visual foresight and actions conditioned on both current and historical observations, making them a promising paradigm for robotic manipulation. However, existing WAMs face a fundamental trade-off: methods with efficient inference typically condition only on a bounded window of recent observations and therefore struggle in non-Markovian environments, whereas methods that preserve long histories incur time and space costs that grow substantially with sequence length. To address this challenge, we introduce MemoryWAM, a world action model with efficient persistent memory. MemoryWAM uses a hybrid memory design that combines recent frames, event-boundary anchor frames, and compact gist tokens that summarize long-range history. A tailored attention mechanism enables retrieval of both detailed short-term context and compressed long-term context, supporting memory-dependent decision-making with reduced inference latency and GPU memory usage. Across long-horizon, memory-dependent manipulation tasks in both simulation and the real world, MemoryWAM outperforms strong vision-language-action (VLA) and WAM baselines while maintaining favorable computational efficiency.

## 1. 核心问题与整体含义（研究动机与背景）

- **核心问题**：机器人操作需要同时处理当前观测、历史记忆和物理动力学建模。世界动作模型（WAM）通过联合建模视觉预测和动作条件具备这些能力，但现有WAM面临**记忆-效率的根本权衡**：
  - 高效推理方法（如滑动窗口）只能利用有限近期的观测，在非马尔可夫（状态不满足马尔可夫性）环境中失败，因为关键信息可能超出窗口范围。
  - 保留完整历史的方法（如全KV缓存）虽然提供长程上下文，但推理延迟和GPU内存随序列长度线性增长，难以用于长时域任务。
- **整体含义**：该工作旨在为WAM设计一种**高效持久记忆**机制，在不牺牲性能的前提下大幅降低推理代价，使机器人能够处理需要长期记忆的复杂操作任务（如物体被遮挡后需要记忆位置、根据历史计数执行动作等）。

## 2. 方法论

### 2.1 核心思想

- 受人类认知的**混合记忆系统**启发（短期记忆、长期记忆中的“要点”痕迹、事件边界记忆），MemoryWAM提出**混合记忆设计**：
  - **短期记忆**：滑动窗口保留最近高保真帧，支持即时闭环控制。
  - **事件边界记忆**：保留任务初始帧的完整视觉token（锚帧），因为初始场景信息常被任务指令引用，且可能被遮挡或移出窗口。
  - **要点记忆**：为每个历史帧附加**M个可学习的gist token**（M ≪ L，L为每帧视觉token数），将长程历史压缩为紧凑表示，后续查询只与gist token交互，而非完整token。

### 2.2 关键技术细节

- **架构**：基于Mixture-of-Transformers (MoT) 范式：
  - **视频专家**：使用预训练的Wan2.2-TI2V-5B视频DiT（30层，隐藏维3072，24头，头维128），负责视觉动力学建模与记忆管理。
  - **动作专家**：独立的动作DiT（30层，隐藏维1024，24头，FFF维4096，约1B参数），通过去噪预测动作块（每次16步动作）。动作DiT的权重通过沿隐藏维插值从视频DiT初始化。
  - 训练时视频预测提供密集的动力学监督，推理时**不生成视频**，仅使用干净帧的KV缓存。
- **混合记忆注意力掩码**：
  - 近期帧和锚帧：保留完整KV token，后续动作查询可直接访问。
  - 历史非关键帧：只保留其gist token的KV；每个历史帧的视觉token被丢弃。
  - 每个gist token通过自注意力从其对应帧的视觉token和历史上下文中聚合信息。
- **复杂度分析**：
  - 设每帧视觉token数L=120，gist token数M=8，压缩比d = L/M = 15。
  - 全历史记忆：O(NL)；混合记忆：O(NM) = O(NL/d)，实现15倍KV缓存缩减。
- **推理协议**：每步维护混合KV cache：
  - 前Ninit=2个锚帧（完整token）
  - 最近Nrecent=4个干净帧（完整token）
  - 所有帧的M=8个gist token（永不被清除）
- **训练**：连续流匹配，1000步噪声调度，视频和动作分支分别使用shift logit-normal分布。使用自强制（Self-forcing）技巧对干净帧加噪防止过拟合。损失函数L = λv Lvideo + λa Laction，λv=λa=1.0。
