## 摘要
物体交互任务一直是模仿学习进展的焦点。以扩散和流变体为主的端到端方法在性能上取得了飞跃，但牺牲了可解释性。以物体为中心和考虑姿态的变体在操纵任务中的示范学习方面发挥了作用。在本文中，我们重新审视了几个用于物体交互的现代模仿学习基准，旨在构建一个框架，重新利用操纵的原理理论，同时保持性能和可解释性。对于图像观察，我们提出了轻量级训练，用于目标物体的无模型姿态估计，利用模仿学习中可用的示范数据进行自监督。然后，这些信息用于指导一个操纵模式感知的示范检索、一个模式感知变换、一个重新规划步骤（该步骤在保持模式约束的同时连接到检索点），最后执行变换后的示范。这些构成了所提出的RE4框架的四个关键步骤，并在Push-T和Robomimic中的基于状态和基于图像的基准上进行了评估。一个评估基于图像的Push-T稀疏数据区域的对抗性基准展示了鲁棒性，低数据区域实验的指示进一步支持了这一点。目前的工作显示了使用简单可解释的构建模块来学习操纵技能的前景。

## Abstract
Object interaction tasks have been a focus of advances in imitation learning. End-to-end methods, dominated by diffusion and flow-based variants have shown leaps in performance while sacrificing interpretability. Object-centric and pose-informed variants have had a role in learning from demonstration in manipulation tasks. In this paper, we revisit a few modern imitation learning benchmarks for object interactions, with the aim of composing a framework that repurposes principled theories of manipulation, preserving both performance and interpretability. For image observations, lightweight training is proposed for model-free pose estimation of the target object, using self-supervision over the demonstration data available for imitation learning. This information is then used to inform a manipulation mode-aware retrieval of a demonstration, a mode-aware transformation, a replan step that connects to the retrieval point while preserving mode constraints, and finally rolling out the transformed demonstration. These compose four key steps of the proposed RE4 framework, evaluated over state-based and image-based benchmarks in Push-T and Robomimic. An adversarial benchmark that evaluates sparse data regions of image-based Push-T showcases the robustness, further bolstered by indications from low-data regime experiments. The current work shows promise in using simple interpretable building blocks to learn manipulation skills.

## 二、论文提出的方法论：核心思想、关键技术与流程

### 核心思想
- 将演示轨迹表示为**模式增强的轨迹**（mode-augmented trajectory），包含末端执行器位姿、物体位姿和操控模式（transit/transfer）。
- 在推理时，根据当前观测和机器人状态，推断操控模式和物体姿态，然后检索最近邻的演示帧，通过**物体中心重框架**将演示变换到当前物体坐标系下，再用**运动规划**连接当前状态与变换后的起始点，最后执行变换后的演示动作序列。

### 关键技术细节
1. **姿态估计**（GetPose）：
   - 对于抓取类任务（prehensile）：提出**自监督学习**方法，仅用演示数据（无真实物体姿态、无CAD模型）训练一个轻量CNN（ResNet-18 + 线性头），输出SE(3)位姿。损失函数包含绝对回归、同一演示内相对一致性、跨演示一致性三项。
   - 对于非抓取类任务（如Push-T）：使用颜色阈值分割加对数极坐标变换（log-polar transform）进行封闭形式的姿态估计（无需学习）。
2. **检索**（Retrieve）：
   - 两阶段查找：先对每个演示计算当前查询（物体增强状态）的最小距离，得到每演示一帧；再跨演示选择物体姿态距离最小的那一帧。操控模式作为硬过滤器，只允许同模式帧。
3. **重框架**（Reframe）：
   - 根据当前物体姿态和检索到的演示帧，计算物体相对变换δ，然后在transit模式下用δ变换演示动作，在transfer模式下用δ⁻¹保持抓取变换不变，生成针对当前场景的演示动作块。
4. **重规划**（Replan）：
   - 使用运动规划（RRT或闭环伺服）生成一条从当前状态到变换后起始点的无碰撞路径，同时满足模式约束（transit时保持物体静止，transfer时保持抓取变换不变）。
5. **重放**（Replay）：
   - 依次执行重规划路径和重框架后的动作块。

## 六、论文的主要结论与发现

- RE4在性能上匹配或超越现有生成式/参数化基线，尤其在图像任务上优势明显（Push-T图像覆盖0.883 vs. 扩散0.826 vs. GPI 0.624）。

- 在稀疏观测和稀疏演示条件下，RE4性能下降幅度远小于扩散策略，表现出更强的鲁棒性。
- 轻量级自监督姿态估计（仅用演示数据）足以支持RE4框架，无需通用姿态估计或CAD模型。
- 使用简单可解释模块即可实现与复杂黑箱模型相当甚至更好的操纵技能学习。

## 七、优点

- **可解释性强**：每一步（检索、重框架、重规划、重放）都有明确几何意义，可追溯决策依据。
- **训练轻量**：仅图像任务需训练一个轻量姿态估计网络，状态任务无需训练；相比扩散策略数小时训练，RE4只需数分钟。
- **鲁棒性好**：在稀疏数据下表现突出，对抗性初始状态和少演示场景均能维持较高性能。
- **模块化设计**：各组件可独立改进或替换，便于后续扩展。
- **理论基础扎实**：基于操控模式（transit/transfer）和物体中心变换，避免了黑箱学习的不稳定性。

## 八、不足与局限

- **姿态估计依赖**：姿态估计误差会传播到下游所有步骤；在非抓取任务中姿态估计更困难，当前仅针对特定任务设计闭环解法，未提出通用非抓取姿态估计方法。
- **物体种类有限**：仅在Push-T和Robomimic的三个物体（T块、物块、罐、方形）上验证，未覆盖多样物体和材质。
- **多物体场景未处理**：任务只涉及单个目标物体，对多物体交互需要枚举和管理模式，尚未支持。
- **仅接触式操作**：未涉及更丰富的可变形物体或非接触式操作（如推、滑动等）。
- **缺乏真实机器人验证**：所有实验均在仿真中进行，真实环境下的泛化性和鲁棒性有待验证。
- **模式假设较强**：基于预先定义的transit/transfer模式，对于更复杂或模糊的操作模式可能需要扩展。
