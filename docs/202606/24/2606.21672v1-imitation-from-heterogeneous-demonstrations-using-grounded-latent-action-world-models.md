## 摘要
模仿学习已成为学习视觉运动策略的强大范式，但其泛化能力和稳定性受到所需示范数据的规模和质量限制。一个有前景的方向是利用更丰富但异构的数据源，这些数据源在动作空间上不同，并且通常完全缺乏动作标签。现有的结合异构数据源的联合训练方法依赖于启发式和手工设计的对齐技术。相比之下，我们主张动作表示应该基于预测：在环境中产生相同效果的动作应共享相同的表示，无论其来源如何。为此，我们通过使用基于基础潜动作世界模型（GLAM）来实例化这一原则，该模型是一对生成模型，跨数据源共享一个潜在动作空间，并通过跨源一致地预测未来观察结果来将其基础化。这个潜动作空间用于训练下游的行为克隆（BC）策略，该策略将观察结果映射到潜动作，并将其解码回机器人动作，从而提供了一种从异构数据中学习的范式。实验结果表明，GLAM成功学习了一个对齐的潜动作空间，促进了有动作标签和无动作标签的数据源之间的动作转移。在仿真和现实世界的五个操作任务中，GLAM对齐的策略显著优于BC基线和先前的潜动作方法，在相同数据稀缺设置下平均任务成功率提高了48%。视频和代码可在 https://viccccciv.github.io/glam/ 获取。

## Abstract
Imitation learning has emerged as a powerful paradigm for learning visuomotor policies, but its generalisation and stability are limited by the scale and quality of demonstration data needed. A promising direction is to leverage more abundant but heterogeneous data sources, which differ in action space and often lack action labels altogether. Existing co-training approaches that combine heterogeneous data sources rely on heuristic and hand-engineered alignment techniques. In contrast, we argue that action representations should be grounded in prediction: actions that produce the same effect on the environment should share the same representation, regardless of their sources. To this end, we instantiate this principle by using a grounded latent-action world model (GLAM), a pair of generative models with a shared latent action space across data sources that is grounded by predicting future observations consistently across sources. This latent action space is used to train downstream behavioural cloning (BC) policies which map observations to latent actions and decode them back to robot actions, providing a paradigm for learning from heterogeneous data. Empirically, we demonstrate that GLAM successfully learns an aligned latent action space that facilitates action transfer across data sources with and without action labels. Across five manipulation tasks in simulation and in the real world, GLAM-aligned policies significantly outperform BC baselines and prior latent-action methods, achieving an average of +48% improvement in task success rate with the same data-scarce setting. Videos and code are available at https://viccccciv.github.io/glam/.

- **核心问题**：现代模仿学习虽在视觉运动操控策略上表现优异，但其泛化能力严重依赖于大规模、高质量的示范数据。行为克隆（BC）策略通常需要数百至数千条在目标机器人上遥操作的轨迹才能达到可靠部署的阈值，而收集这些数据成本高昂且缓慢，成为扩展实用策略的主要瓶颈。
- **研究动机**：利用更廉价、更丰富的异构数据源（如便携手持设备采集的轨迹、仿真数据、人类视频等）来补充目标机器人示范数据，具有巨大潜力。但这些异构源的动作空间不同，往往缺乏动作标签，且存在视觉外观、场景和具身上的显著差异，简单混合训练通常无效。
- **关键洞察**：本文主张动作表示应基于对环境的**效果**进行“接地（grounding）”——两个动作若使物体沿相同轨迹运动，无论其来源（不同具身、真实或仿真），应被相似表示。这可将跨源集成转化为表示学习问题，以环境转移作为共享的物理接地监督信号。

## 2. 方法论：核心思想、关键技术细节与流程
### 2.1 核心思想
- 利用**世界模型**（world model）预测动作如何塑造环境转移，为潜动作提供自然的物理接地。将动作视为潜变量，构建一对耦合生成模型（GLAM）：一个用于全部异构数据（目标+辅助），另一个仅用于带标签的目标数据。
- 通过共享的前向动力学（forward dynamics）以及**不对称KL散度对齐**，将两个推理路径（IDM和动作编码器）绑定到同一对齐潜动作空间。

### 2.2 关键技术细节
#### 阶段1：GLAM预训练
- **异构生成模型**：建立潜空间动力学模型，对观测序列建模，使用**逆动力学模型（IDM）** `q_ϕ(z_t | x_t, x_{t+1})` 从状态转移推断潜动作，共享前向动力学 `p_θ(x_{t+1} | x_t, z_t)`，保证源不变性。优化ELBO（公式3）。
- **目标生成模型**：额外利用机器人状态和动作标签，使用**动作编码器** `q_η(z_t | s_t, a_t)`，并加入动作重建项 `log p_θ(a_t | z_t)`，注入控制感知信息。优化ELBO（公式4）。
- **不对称对齐**：对目标数据集的部分，引入KL对齐约束 `C_KL`，使IDM后验与动作编码器后验对齐（对动作编码器使用stop-gradient），确保IDM吸收目标动作语义，同时保持源不变性。
- **总目标**：联合优化两个负ELBO与对齐约束（公式6），超参数λ=1。
- **物体掩码观测**：为强化“动作效果应基于物体运动”的假设，实验中使用预训练分割模型提取的**二进制物体掩码**替代RGB图像（称为GLAM-O变体）。

#### 阶段2：世界模型对齐的BC
- 冻结训练好的GLAM，使用IDM后验均值重新标记所有数据（目标+辅助）的潜动作 `z_t`，形成大规模（观测→潜动作）对。
- 下游BC策略（采用MIP [3] 架构）预测潜动作块，再通过动作解码器 `h_a` 解码为可执行机器人动作。训练目标（公式8）包含两项：所有数据上的潜动作L2损失 + 目标数据上的原始动作L2损失。

### 2.3 算法流程（文字说明）
1. 采集少量目标示范（含动作标签）和大量辅助示范（无动作标签）。
2. 在目标+辅助混合数据上预训练GLAM（IDM、动作编码器、前向模型、重建模型等），联合优化ELBO+对齐损失。
3. 冻结GLAM，用IDM给每个转移标记潜动作。
4. 训练潜动作BC策略，以观测为输入，预测潜动作块，解码为机器人动作，端到端优化。
