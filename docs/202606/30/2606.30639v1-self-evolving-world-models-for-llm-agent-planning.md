## 摘要
世界模型为长视距LLM智能体提供了一种有原则的方式来赋予其预见能力：在执行之前预测行动后果。然而，不可靠的预见可能被忽略、误用，甚至降低下游决策的质量。在本文中，我们引入了WorldEvolver，这是一个自我进化的世界模型框架，它在保持下游智能体和所有模型参数冻结的同时，修改其部署时的上下文。WorldEvolver整合了三个模块：（i）情景记忆，通过基于检索的模拟利用真实行动转换；（ii）语义记忆，从预测-观察不匹配中提取持久启发式规则；（iii）选择性预见，在将预测整合到智能体推理上下文之前过滤低置信度预测。我们在ALFWorld和ScienceWorld上评估WorldEvolver，在Word2World上测量世界模型预测准确率，在AgentBoard上测量下游智能体成功率。大量实验表明，WorldEvolver在三个主干网络上实现了最高的预测准确率，并在下游智能体成功率上领先其他世界模型基线，证明测试时记忆修订增强了预测保真度和规划性能。

## Abstract
World models offer a principled way to equip long-horizon LLM agents with foresight: predictions of action consequences before execution. However, unreliable foresight can be ignored, misused, or even degrade downstream decision-making. In this paper, we introduce WorldEvolver, a self-evolving world model framework that revises its deployment-time context while keeping the downstream agent and all model parameters frozen. WorldEvolver integrates three modules: (i) Episodic Memory, which exploits real action transitions through retrieval-based simulation; (ii) Semantic Memory, which extracts persistent heuristic rules from prediction-observation mismatches; and (iii) Selective Foresight, which filters low-confidence predictions before integrating them into agent reasoning context. We evaluate WorldEvolver on ALFWorld and ScienceWorld, measuring world model prediction accuracy on Word2World and downstream agent success rate on AgentBoard. Extensive experiments show that WorldEvolver achieves the highest prediction accuracy across three backbones and leads other world model baselines on downstream agent success rate, demonstrating that test-time memory revision enhances both predictive fidelity and planning performance.

## 1. 论文的核心问题与整体含义（研究动机和背景）

- **核心问题**：LLM智能体在执行长视距任务时，需要世界模型提供对行动后果的预见（foresight），但现有世界模型的预测往往不可靠，易被忽略、误用，甚至损害下游决策。如何在不更新模型参数（冻结智能体与世界模型）的前提下，让世界模型在部署过程中自我进化、提高预测质量，是本文要解决的关键问题。
- **研究动机**：分布偏移（sim-to-real gap）导致冻结世界模型预测失准；在线参数更新（梯度更新）计算成本高且可能引发灾难性遗忘。因此，需要一种**测试时无参数更新的自我进化机制**，通过记忆而非权重修改来适应环境变化。
- **整体含义**：提出WorldEvolver框架，通过持续修订部署时的上下文（记忆内容），而非更新模型参数，使世界模型的预测更准确，进而提升LLM智能体的规划性能。

## 2. 论文提出的方法论：核心思想、关键技术细节

- **核心思想**：在冻结世界模型 \(W_\theta\) 和冻结智能体策略 \(\pi_\theta\) 的前提下，通过外部非参数记忆的在线演化来改进预见质量。记忆分为两类（Episodic Memory 和 Semantic Memory），并结合选择性预见机制过滤不可靠预测。
- **关键技术细节**：
    - **情景记忆（Episodic Memory, ME）**：存储已执行的状态-动作-下一状态三元组 \(\{(o_i, a_i, o_{i+1})\}_{i<t}\)。给定候选动作 \(a_t\)，通过动作令牌的Jaccard相似度检索 \(k_{ME}\) 个最相似的历史转换，以RAW文本形式作为上下文提供给世界模型，用于预测。记忆随执行逐步追加。
    - **语义记忆（Semantic Memory, MS）**：从预测-观察失配中提取持久启发式规则。将原始观察文本通过映射函数 \(g\) 转化为因式化三元组（subject, predicate, object），比较预测与真实观察的三元组差异。若失配，由LLM批评者生成文本规则，每条规则附带证据得分 \(e_i\)，初始为1，后续根据支持/矛盾以 \(\pm 1/|M_S|\) 更新。仅保留证据分>0的规则。可累积小批量（\(k_{MS}\)）失配后批量更新。
    - **选择性预见（Selective Foresight）**：计算预测输出的几何平均令牌概率 \(q_t = \exp(\frac{1}{n}\sum_i \log p_\theta(y_i|\dots))\)，若 \(q_t \ge \tau\) 则向智能体提供预测 \(\hat{o}_{t+1}\)，否则不提供（置空 \(\emptyset\)）。以此过滤低置信度预测，避免误导智能体。
- **算法流程**（文字说明）：
    1. 智能体从当前状态采样草稿动作 \(a_t^{(0)}\)。
    2. 从Episodic Memory检索 \(k_{ME}\) 个历史转换。
    3. 世界模型在包含记忆的上下文中预测观察 \(\hat{o}_{t+1}\) 并计算置信度 \(q_t\)。
    4. 若 \(q_t \ge \tau\)，将预测作为Foresight \(F_t\) 传入智能体；否则 \(F_t = \emptyset\)。
    5. 智能体基于 \((s_t, F_t)\) 采样最终执行动作 \(a_t\)。
    6. 若最终动作与草稿不同，再次用 \(a_t\) 查询世界模型以获取对齐的预测 \(\hat{o}_{t+1}\)。
    7. 执行动作 \(a_t\)，观察 \(o_{t+1}\)。
    8. 更新Episodic Memory：追加 \((o_t, a_t, o_{t+1})\)。
    9. 若预测与观察失配，通过LLM批评者提取规则，更新Semantic Memory。

## 8. 不足与局限

- **评估范围有限**：仅在ALFWorld和ScienceWorld两个文本规则环境测试，未涉及真实网页导航（WebArena, Mind2Web）、机器人控制、多模态环境。环境的状态/动作空间结构化且规模有限，泛化性存疑。
- **置信度估计依赖token概率**：该方法需要模型输出token概率，对于封闭API模型（如GPT-4封闭接口）可能不可用。替代方案（自一致性、校准模型）未被探索。
- **记忆机制局限性**：情景记忆使用简单的Jaccard相似度，在大型或高维动作空间中可能不够鲁棒；语义记忆的规则提取依赖LLM批评者质量，可能产生噪声规则或遗漏关键规则。证据得分更新机制为手工设计（\(\pm 1/|M_S|\)），缺乏自适应。
- **计算开销**：WorldEvolver的运行时虽只略高于Zero-Shot（例如Gemma-4-26B-A4B从1.05s到1.48s），但每次执行需两次模型调用（草稿预测+最终动作预测），并配备检索与规则提取步骤，在大规模部署时成本仍需关注。
- **消融实验不完整**：未对Selective Foresight的不同门控策略（如动态阈值 vs 固定阈值）进行系统比较；未对Semantic Memory中规则抽样的不同记忆代码（证据更新公式）进行消融。
- **基线弱点**：RAWM-\(\phi\)仅使用离线固定检索库，不能在线更新；ITP-I被限制为单步预测，在规划部分却允许多步想象，对比不够一致（论文已说明限制）。更好的基线可能是允许在线记忆的检索增强方法。
